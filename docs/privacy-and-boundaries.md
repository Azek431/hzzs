# 隐私与边界

本文档明确 HZZS 的功能边界、数据处理原则和隐私承诺。

## 核心定位

**HZZS 是本地、只读的跑酷画面分析与 HUD 辅助工具。**

这意味着：

- 所有分析计算在用户设备本地完成。
- 不向任何远程服务器发送数据。
- 不收集、不上传、不共享用户的运行记录、截图、日志或设备信息。

| 能力 | 说明 |

## 数据处理原则

### 本地优先

- 所有记录、配置和诊断信息默认保存在设备本地。
- 用户可随时查看、导出或清理本地数据。

### 授权先行

- 涉及屏幕采集时，会通过系统授权流程（MediaProjection）明确要求用户授权。
- 用户可随时撤销授权。

### 透明说明

- 任何数据采集行为都会在 UI 中明确说明用途。
- 不会在后台静默采集或分析。

### 不收集个人信息

- 不收集用户名、手机号、设备 IMEI、MAC 地址等可识别个人身份的信息。
- 诊断日志中不包含用户隐私数据。

## Issue 提交注意事项

在 GitHub 提交 Issue 或 Pull Request 时，请**不要**包含以下内容：

- 手机号
- 账号信息（游戏账号、QQ 号、邮箱）
- 身份证明、住址等个人敏感信息
- 包含隐私的游戏截图（分数、等级、好友列表）
- 签名文件、密钥库密码、私钥密码
- 未脱敏的系统日志（可能包含设备信息）

如果日志中包含敏感信息，请在提交前手动替换为 `***REDACTED***`。
